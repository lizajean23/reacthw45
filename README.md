# Horizon

A React app with two pages — Home and Users.

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Pages

- **Home** (`/`) — Hero section with photo, text, and a button that navigates to the Users page
- **Users** (`/users`) — Data from [JSONPlaceholder API](https://jsonplaceholder.typicode.com/users) (name, username, email, city)

## Tech stack

- React 19
- Vite
- React Router

## Production build

```bash
npm run build
npm run preview
```
